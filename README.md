# Eduvex - E-learning Platform

## Tech Stack
- MongoDB
- Express
- React
- Node.js
- TailwindCSS

## Features
- User Authentication
- Course Purchase
- Instructor Dashboard
- Admin Panel

## How to Run

### Backend
cd eduvex_backend
npm install
npm run dev

### Frontend
cd eduvex_frontend
npm install
npm run dev

### Root
cd
